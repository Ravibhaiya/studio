# **App Name**: Flashy

## Core Features:

- Create Flashcards: Allow users to create flashcards with a term and a definition.
- View Flashcards: Display flashcards, one at a time, allowing the user to flip them to reveal the definition.
- Organize Decks: Enable users to organize flashcards into decks for focused study.

## Style Guidelines:

- Primary color: Light Blue (#ADD8E6) for a calm and focused learning environment.
- Secondary color: Light Gray (#D3D3D3) for backgrounds and subtle accents.
- Accent: Teal (#008080) for interactive elements and important information.
- Clean and minimalist design to reduce distractions.
- Simple, intuitive icons for navigation and actions.
- Subtle transitions when flipping flashcards or navigating between decks.
